## 📖 Overview

**VIRO XYZ** is a dynamic web-based script hub designed for **educational purposes** to help students and developers understand:
- JavaScript automation concepts
- GitHub API integration
- Web development best practices
- Browser console manipulation
- Security and risk analysis

> ⚠️ **IMPORTANT**: This tool is for LEARNING ONLY. Using these scripts on live Blooket games violates Blooket's Terms of Service and may result in permanent account bans.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔄 **GitHub API Integration** | Dynamically fetches scripts from Glixzzy's repository |
| 🔍 **Search & Filter** | Find scripts by name or game mode |
| 📊 **Real-time Stats** | View script count, mode count, and API status |
| 🛡️ **Sandbox Mode** | Safely analyze scripts without executing them |
| 🔎 **Risk Analysis** | Detects dangerous patterns in scripts |
| 📋 **Copy to Clipboard** | One-click script copying |
| 💾 **Download Scripts** | Save scripts as .js files |
| 🎨 **Modern UI** | Purple gradient with halftone grid background |
| 📱 **Responsive Design** | Works on desktop, tablet, and Chromebook |
